Praticando com o Git e Github

O que é o GIT?

Git é um sistema de controle de versões distribuídos (SCV) usado no desenvolvimento de softwares, cuja a função é registrar quaisquer alterações feitas em cima de um código, possuindo três estados sendo eles estado modificado, estado preparado e estado modificado.

O que é o Github

GitHub é uma plataforma totalmente online onde você pode criar repositórios e hospedar neles seus projetos, colaborar com softwares , seguir outros programadores e interagir com códigos de terceiros, ou seja, é como uma rede social para programadores.
